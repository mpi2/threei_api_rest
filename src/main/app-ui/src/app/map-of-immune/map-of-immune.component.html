Map of the Immune System

This map of the mouse immune system was built using data from over 400 C57BL/6N wild type mice. Each rectangle represents an immune cell subset. The numbers are percentages of CD45+ cells. T cell subsets are blue, B cells orange, Myeloid cells green, NK cells red and NKT cells purple. Use the interactive legend on the top right to switch between organs and cell subsets.
